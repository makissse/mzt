====================================================

Implement the following custom 90-point scoring system.

Base criteria (1–10):

1. Rhymes / Imagery
2. Structure / Rhythm
3. Style Execution
4. Individuality / Charisma

Multiplier (1–10):

Atmosphere / Vibe

Calculation:

Step 1:

(base1 + base2 + base3 + base4)

×

1.4

Step 2:

Multiply the result using:

1 → 1.0000

2 → 1.0675

3 → 1.1349

5 → 1.2699

6 → 1.3373
